Terminal Reader

Distraction-free reading environment.

01. Overview

Features

  • Fullscreen terminal reader
  • Folder-based library browser
  • Whole-word search with highlight
  • TOC support (where available)
  • Reading position persistence

Important Notes

  • TOC: epub (nav/NCX), fb2 (XML), pdf (heuristics)
  • Footnotes: epub supported ([1], ¹²)
  • Position: saved automatically per file

Supported Formats

txtfb2epubpdf

Dependencies

  • fb2 / epub: pandoc
  • pdf: pdftotext (from poppler)
